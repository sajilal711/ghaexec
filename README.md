# RDK Curl Generator

This repo is now a single-file, frontend-only curl generator.

## File

- `rdk-build-trigger-generator.html`

## What it does

- Builds the JSON payload
- Generates a curl command
- Lets you copy curl and JSON
- Lets you download JSON

## What it does not do

- No proxy support
- No backend executor
- No direct API trigger button

## Usage

1. Open `rdk-build-trigger-generator.html` in your browser.
2. Fill inputs and patchsets.
3. Copy the generated curl command.
4. Run curl manually in terminal.
